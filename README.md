# Digital Aragão R05

Projeto React + Vite com múltiplas páginas e visual premium.

## Rodar localmente

```bash
npm install
npm run dev
```

## Build de produção

```bash
npm run build
npm run preview
```

## Deploy no Render

- Suba a pasta no GitHub
- Crie um novo Web Service / Static Site no Render
- O arquivo `render.yaml` já está incluído
